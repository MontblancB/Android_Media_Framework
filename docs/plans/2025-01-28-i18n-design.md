# 다국어(영문) 지원 설계 문서

**작성일:** 2025-01-28
**상태:** 승인됨

## 개요

Android Media Framework 문서 사이트에 영문 버전을 추가하여 글로벌 팀(LGE 해외 법인/파트너)과 공유할 수 있도록 한다.

## 결정 사항

| 항목 | 결정 |
|------|------|
| URL 구조 | `/` = 한글 (기존 유지), `/en/` = 영문 |
| 번역 범위 | 전체 40개 페이지 |
| 번역 방식 | AI 초벌 번역 + 검토 (하이브리드) |
| UI | 헤더에 `[KO | EN]` 토글 버튼 (다크모드 토글 왼쪽) |
| 기본 언어 | 한글 (루트 URL 유지) |

## 폴더 구조

```
Android_Media_Framework/
├── index.html                 # 한글 메인 (기존 유지)
├── aosp.html                  # 한글 페이지들 (기존 유지)
├── ...
│
├── en/                        # 영문 버전 폴더 (신규)
│   ├── index.html             # 영문 메인
│   ├── aosp.html
│   └── ...
│
├── styles/
│   └── design-system.css      # 언어 전환 버튼 스타일 추가
│
└── scripts/
    ├── diagram-data.js        # 기존 유지
    ├── theme-toggle.js        # 기존 유지
    └── lang-switch.js         # 언어 전환 로직 (신규)
```

## 언어 전환 버튼 UI

**위치:** 헤더 우측, 다크/라이트 모드 토글 왼쪽

```
┌─────────────────────────────────────────────────────┐
│  ← Back to Home          [KO | EN]  [🌙/☀️]        │
└─────────────────────────────────────────────────────┘
```

**동작:**
- 한글 페이지에서 `EN` 클릭 → `/en/현재페이지.html`로 이동
- 영문 페이지에서 `KO` 클릭 → `/현재페이지.html`로 이동
- 현재 언어는 활성화 스타일 (accent 색상)

## 번역 원칙

1. **기술 용어는 원어 유지**
   - MediaPlayer, Codec2, AudioFlinger, HAL, Binder 등

2. **UI 텍스트 영문화**
   - ← Back to Home, Explore → 등

3. **AOSP 공식 문서 용어와 일치**
   - Android 공식 영문 문서의 용어 사용

4. **Mermaid 다이어그램**
   - 노드 텍스트도 영문화
