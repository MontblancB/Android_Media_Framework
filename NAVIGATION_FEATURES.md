# 🎉 내비게이션 및 복사 기능 추가 완료!

## 📋 추가된 기능

### 1️⃣ **코드 복사 버튼**
모든 코드 블록에 자동으로 복사 버튼이 추가됩니다.

**기능**:
- 📋 클릭 한 번으로 코드 복사
- ✅ 복사 성공/실패 피드백
- 🏷️ 자동 언어 감지 (Kotlin, Java, JavaScript, Python, Shell 등)
- 📱 모바일 최적화

**사용 방법**:
1. 코드 블록 위의 [📋 복사] 버튼 클릭
2. [✅ 복사됨!] 피드백 확인
3. 에디터에 붙여넣기 (Ctrl+V / Cmd+V)

---

### 2️⃣ **페이지 내 목차 (TOC)**
긴 페이지에서 섹션 간 빠른 이동이 가능합니다.

**기능**:
- 📑 오른쪽 사이드바에 목차 자동 생성
- 🎯 현재 읽고 있는 섹션 자동 하이라이트
- ⚡ 클릭 시 스무스 스크롤
- 📱 모바일에서는 자동 숨김

**위치**: 화면 오른쪽 상단 고정

---

### 3️⃣ **Breadcrumb 네비게이션**
현재 위치를 한눈에 파악할 수 있습니다.

**구조**:
```
🏠 Home / Media Framework / Codec 2.0
```

**기능**:
- 클릭 가능한 경로 표시
- 카테고리별 분류
- 현재 페이지 강조

---

### 4️⃣ **이전/다음 페이지 버튼**
25개 페이지를 순차적으로 탐색할 수 있습니다.

**위치**: 각 페이지 하단 (Footer)

**구조**:
```
┌────────────────────────────────────────────────────────┐
│  ← 이전 페이지           🏠 메인으로      다음 페이지 →  │
│  Media Framework Core                  Dolby Codecs    │
└────────────────────────────────────────────────────────┘
```

**기능**:
- 이전 페이지로 이동
- 메인 페이지로 돌아가기
- 다음 페이지로 이동
- 호버 시 애니메이션 효과

---

## 🧪 테스트 방법

### 로컬 서버 실행

```bash
# Python 3 내장 서버 (권장)
python3 -m http.server 8001

# 또는 Node.js live-server
npx live-server --port=8001
```

### 브라우저 테스트

1. **브라우저 열기**: http://localhost:8001

2. **메인 페이지에서 아무 카드나 클릭** (예: Card 5 - Codec 2.0)

3. **코드 복사 버튼 테스트**:
   - 코드 블록 찾기
   - [📋 복사] 버튼 클릭
   - [✅ 복사됨!] 확인
   - 메모장에 붙여넣기 테스트

4. **목차(TOC) 테스트**:
   - 오른쪽 사이드바 확인
   - 목차 항목 클릭 → 스무스 스크롤 확인
   - 스크롤 시 활성 섹션 하이라이트 확인

5. **Breadcrumb 테스트**:
   - 상단 네비게이션 확인
   - `🏠 Home / Media / Codec 2.0` 형식 확인
   - 각 링크 클릭 가능 확인

6. **이전/다음 버튼 테스트**:
   - 페이지 하단 확인
   - [← 이전 페이지] 클릭 → Media Framework Core 이동
   - [다음 페이지 →] 클릭 → Dolby Codecs 이동
   - [🏠 메인으로] 클릭 → index.html 이동

## 🐛 알려진 이슈

### TOC가 보이지 않는 경우
- **원인**: 화면 너비가 1200px 미만
- **해결**: 데스크톱에서 확인 (모바일에서는 자동 숨김)

### 코드 복사 버튼이 중복되는 경우
- **원인**: 스크립트 중복 로드
- **해결**: HTML에서 `copy-code.js`가 한 번만 로드되는지 확인

### 이전/다음 버튼이 없는 경우
- **원인**: `page-navigation.js`에 페이지가 등록되지 않음
- **해결**: `scripts/page-navigation.js`의 `PAGE_ORDER` 배열에 추가
